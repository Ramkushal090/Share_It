# Share_It
A campus-wide item sharing platform where students can list items they own and borrow items from others for a selected number of days. Built using Spring Boot and Maven.

ShareIt is a simple platform that allows students on a campus to **list items they own** and **borrow items from others** for a selected number of days. The goal is to reduce unnecessary purchases and make resource sharing easier.

## Features
- List items with name, description, and availability  
- Browse and search items available on campus  
- Borrow items for a chosen number of days  
- Track borrowed and lent items through a dashboard

- Maven
